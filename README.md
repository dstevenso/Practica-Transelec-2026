# Practica-Transelec-2026
Archivos asociados a actividades en Transelec durante la práctica en 2026

DTW_Vibraciones.py : Prueba de concepto para análisis transiente de vibraciones.

Medidor.cpp : Script que mide vibraciones después de superar un cierto umbral y hace un envio binario.

Umbral_falla.py : Prueba de concepto para detectar potencial falla mediante umbral en espectro de frecuencia.

Wavelet.py : Prueba de concepto de análisis transiente y de frecuencia para señales multi frecuencia.
